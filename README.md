## 구성
각 연도 대시보드는 다음을 시각화합니다.
- CCS 분야 분포 (ACM 최상위 분류)
- 저자 키워드 Top 20 / 워드클라우드
- Main(풀페이퍼) vs Companion 트랙 비교
- 분야·키워드 클릭/검색 → 논문 목록 + Abstract + DOI

## 저작권
표시되는 서지정보·Abstract는 ACM Digital Library 공개 메타데이터 기반이며 권리는 각 저자/ACM에 귀속됩니다.
논문 전문은 각 항목의 DOI를 통해 ACM에서 열람할 수 있습니다. 비상업적 학술 리뷰·교육 목적입니다.
